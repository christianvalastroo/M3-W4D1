# 🎬 Netflix Clone – Progetto Bootstrap

Questo progetto consiste nella realizzazione di una homepage stile Netflix sviluppata utilizzando HTML, CSS e Bootstrap 5.  
L'obiettivo era ricreare l'interfaccia di una piattaforma di streaming con righe di film, barra di navigazione e footer mantenendo una struttura responsive e moderna.

---

## 🚀 Funzionalità

- Layout responsive realizzato con Bootstrap Grid
- Navbar stile Netflix con logo, menu e icone
- Sezioni di film organizzate in categorie:
  - Trending Now
  - Watch It Again
  - New Releases
- Scroll orizzontale delle card su dispositivi mobili
- Effetti hover sulle card dei film
- Animazione delle sezioni quando entrano nello schermo durante lo scroll
- Footer stile Netflix con icone social e link informativi

---

## 🛠️ Tecnologie utilizzate

- HTML5
- CSS3
- Bootstrap 5
- Bootstrap Icons
- JavaScript

## 📱 Responsive Design

Il layout si adatta automaticamente ai diversi dispositivi:

Smartphone  
→ scroll orizzontale delle card

Tablet  
→ griglia con meno colonne

Desktop  
→ 6 card per riga

## 🎨 Stile e design

Il progetto utilizza variabili CSS personalizzate per mantenere la palette di colori tipica di Netflix.

Esempio di variabili utilizzate:

--netflix-bg  
--netflix-red  
--netflix-text-primary  
--netflix-text-secondary  

Le sezioni della pagina vengono animate allo scroll utilizzando IntersectionObserver.
